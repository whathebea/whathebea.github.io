What I have been studying

2022-10-12
1 minute read
tech
java , spring , angular

I believe that, nowadays, writing a blog is just one way to make myself aware that I am studying a lot. Maybe I will talk a lot about myself, but the I.T field can sometimes be very harsh on you (or me). I am aware that, being an intern, I don’t have the same hardships as regular employees, but it can still be hard. I also feel like there are a lot of commas, but to be honest, I never properly learned how to use commas in English and, that being said, Grammarly is correcting it all for me.

I have been on my current internship for 3 months and these are some of the courses I have taken:

  • JDBC Servlets and JSP - Java Web Development Fundamentals
  • Java Enterprise Edition 8 for Beginners course
  • Maven Crash Course
  • Spring Framework In Easy Steps
  • Design Patterns in Java
  • Enterprise Design Thinking
  • Agile 
  • Problem Solving

The ones that I linked to the certificates are the ones I feel I learned the most. I didn’t enjoy the JEE course and that’s why I’ve been reading a lot of the docs.

Here are some of the links that I have found useful so far:

  • Jakarta EE 8 Docs
  • Jakarta Annotations
  • Spring Unit Testing
  • Spring Web Servlets
  • Spring Data

Spring Documentation I found very thorough, to be honest.

Besides a lot of Java, I am also learning Angular and I’ve started the course Angular - The Complete Guide which I believe will be very helpful. I think I have a good grasp of HTML, CSS, and Javascript already, so it won’t be too hard. And Typescript is a strongly typed object-oriented language. It almost sounds like something I have been studying a lot, doesn’t it?

Today is Children’s Day here in Brazil (a holiday), so I won’t be studying. I hope to make a new post tomorrow. 

Thank you for reading!
